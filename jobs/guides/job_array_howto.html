Job Array Howto¶

In this example we wish to run many similar sequential jobs in parallel using job arrays. We take Python as an example but this does not matter for the job arrays:

include

Download the script: array_test.py (you might have to right-click and select Save Link As... or similar).

Try it out:

$ python array_test.py
start at 15:23:48
sleep for 10 seconds ...
stop at 15:23:58

Good. Now we would like to run this script 16 times at (more or less) the same time. For this we use the following

include

Download the script: array_howto.sh (you might have to right-click and select Save Link As... or similar).

This is a script for running a normal array job on Saga. It can easily be changed to run on Fram or use a different job type.

Submit the script with sbatch and after a while you should see 16 output files in your submit directory:

$ ls -l output*txt
-rw------- 1 user user 60 Oct 14 14:44 output_1.txt
-rw------- 1 user user 60 Oct 14 14:44 output_10.txt

-rw------- 1 user user 60 Oct 14 14:44 output_3.txt
